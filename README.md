# Библейские игры — Premium redesign

Мобильное веб-приложение с коллекцией командных, словесных и детских игр для Telegram WebApp / браузера.

## Что изменено в этой версии

- Полностью переработано главное меню: премиальные карточки игр, описания режимов, блок «Недавно играли», новая шапка и загрузочный экран.
- Введена единая дизайн-система для всех игр: графитовый фон, тёплые золотые акценты, крупные кнопки, плотные карточки, одинаковые поля ввода и модальные окна.
- Убран визуальный эффект жидкого стекла: `backdrop-filter` удалён из модалок и оверлеев, вместо него используются плотные затемнения и объёмные карточки.
- Улучшена навигация: исправлена проблема, когда `Alias` переопределял общий `goToMainMenu()` и мог ломать возврат в меню/очистку игры.
- Добавлен тактильный отклик Telegram WebApp при выборе игры, если приложение запущено внутри Telegram.
- Улучшены состояния загрузки и ошибки запуска игры: теперь показываются аккуратные карточки вместо простого красного текста.
- Обновлены стили для Alias, Word Search, Bible WOW, Sacred Word, Kids Pairs и Quartet через общую тему, чтобы игры выглядели частью одного дорогого продукта.

## Запуск

Откройте `index.html` на хостинге или локальном сервере. Для полноценной работы Telegram WebApp и внешних API лучше запускать проект через HTTPS-хостинг.
