# Сервер для онлайн‑игры «Квартет» (бесплатно, Cloudflare Workers)

Этот проект у тебя целиком статический (GitHub Pages). Чтобы сделать **онлайн‑комнаты** для «Квартета», проще всего использовать **Cloudflare Workers + Durable Objects** (на бесплатном тарифе хватает для небольших компаний).

Ниже — инструкция, которую можно выполнить **с iOS (Safari)**.

---

## 1) Создай Cloudflare аккаунт и воркер

1. Зайди в Cloudflare → **Workers & Pages** → **Create application** → **Worker**.
2. Назови, например: `bible-quartet`.
3. Открой редактор кода воркера и замени содержимое на код из файла `server/worker.js`.

---

## 2) Включи Durable Objects

В воркере сверху будет вкладка/меню **Settings**.

1. Найди **Durable Objects** → **Add binding**.
2. Binding name: `ROOMS`
3. Class name: `Room`

---

## 3) Поменяй URL сервера в игре

Открой файл:

`games/quartet.js`

и замени строку:

```js
const WORKER_BASE_URL = "https://REPLACE_ME.workers.dev";
```

на адрес твоего воркера (он будет вида `https://имя.workers.dev`).

---

## 4) Задеплой

Нажми **Deploy** (или **Save and deploy**) в Cloudflare.

После этого:
- «Создать комнату» будет создавать **код комнаты**
- остальные подключаются по коду

---

## Ограничения бесплатного тарифа

На бесплатном тарифе Cloudflare есть лимиты по запросам/соединениям — для игры небольшой компанией обычно хватает.
