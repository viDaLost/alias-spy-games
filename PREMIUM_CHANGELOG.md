# Premium changelog

## Дизайн

- Новый главный экран с крупной брендовой шапкой.
- Карточки игр вместо простых кнопок.
- Единая палитра: графит, слоновая кость, золото, тёплые тени.
- Мобильная адаптация под узкие экраны.
- Убран `backdrop-filter`, чтобы дизайн не выглядел как «жидкое стекло».

## UX

- Блок последних трёх игр.
- Более понятные описания каждой игры.
- Единые состояния загрузки, ошибок, модальных окон и кнопок.
- Быстрее воспринимаемые нажатия: короткие transitions, без тяжёлых blur-эффектов.
- Haptic feedback в Telegram при запуске игры.

## Исправления

- `games/alias.js` больше не ломает глобальный выход в меню: теперь вызывает сохранённый `window.appGoToMainMenu`.
- Проверены все JS-файлы через `node --check`.
- Сохранены существующие данные, игры, API URL и структура проекта.
